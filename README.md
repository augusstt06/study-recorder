## Feature

📌 참고할만한 링크들을 입력받아 화면상에 시각화

- 임의의 카테고리를 생성하여 분류 기능
  - css, framework 등등
- 입력된 링크들은 썸네일을 사용해 구분이 용이하게 한다.

📌 최근에 등록한 링크들은 따로 표시
